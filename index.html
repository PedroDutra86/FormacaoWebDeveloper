Títulos voltar

Em uma estrutura HTML, temos várias categorias de títulos, são elas H1, H2, H3, H4, H5 e H6 quanto menor o número dessa tag, maior e mais chamativo é o título, mas muita atenção, só devemos ter uma única tag H1 em nosso documento, pois se trata de um título principal

Parágrafo voltar

Como o próprio nome já diz, um parágrafo é um elemento de composição de texto de uma categoria e importância visual menor, é nele onde ficarão todo o corpo do texto, como boas práticas, essa tag tem que ter uma visibilidade menos chamativa do que o título, aplicando-se geralmente cores menos contrastadas do que os títulos

Listas voltar

As listas são conteúdos voltados a uma maior organização, elas servem pra você categorizar elementos um embaixo do outro, dentro do universo das listas existem as listas ordernadas OL, as listas desordenardas UL, os elementos de uma lista LI entre outras tags

Elementos de Semântica voltar

Nós temos também algumas tags que modificam o testo como a tag b, deixa o texto em negrito, mas a tag strong també deixa o texto em negrito e além disso traz uma carga semântica para os navegadores, que agora saberão que alí tem uma palavra que merece destaque, temos outras tags tabém como SMALL, I, U, MARK, SUB, SUP, BLOCKQUOTE

Tags Atípicas voltar

Aqui, temos algumas tags como FONT, DEL, ABBR, vamos começar com a tag font, ela foi descontinuada a partir do HTML 5, pois sua função era trazer uma fonte para o documento, e nos dias atuais, isso pode ser implementado facilmente com o CSS, já a tag del, serve pra indicar que aquela parte do documento foi excluida, e a abbr, é uma tag de abreviação, onde por padrão elas são estilizadas com um sublinhado pontilhado
